# MCP Servers

这是一个基于 Node.js 的服务器项目。

## 项目结构

```
mcp-servers/
├── src/           # 源代码目录
├── config/        # 配置文件目录
├── tests/         # 测试文件目录
├── docs/          # 文档目录
└── scripts/       # 脚本文件目录
```

## 开发环境要求

- Node.js >= 16
- npm >= 8

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 测试

```bash
npm test
```
